# Installation overview

**Theme:** Overview  
**Who Is It For?** System Administrator

## What is it?

The Installation section covers all steps required to install and configure the Asysco AMT Connector so that OpCon can schedule and monitor jobs in the Asysco AMT Batch environment.

- When setting up the Asysco AMT Connector for the first time in an OpCon environment
- When adding a connector instance for a new or separate Asysco AMT environment
- When reconfiguring connection settings after changes to the AMT web server address or credentials
- Once installed, OpCon can submit, monitor, and retrieve output for all AMT batch jobs from a central location without requiring manual interaction with the AMT environment
- A separate connector instance per AMT environment keeps each environment's configuration isolated, reducing the risk of cross-environment job submissions
